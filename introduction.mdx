Architectury API is an intermediary library for Minecraft mod development targeting **1.19.2**. It provides a unified API surface that abstracts the differences between [Fabric API](https://github.com/FabricMC/fabric) and [Forge API](https://github.com/MinecraftForge/MinecraftForge), letting you write your mod logic once and deploy it on both platforms.

## The Architectury ecosystem

Architectury API is one part of a three-component toolchain. You need all three to build a multiplatform mod.

| Component | Role |
|---|---|
| **Architectury API** | Runtime library: events, registries, networking, platform helpers |
| [Architectury Loom](https://github.com/architectury/architectury-loom) | Gradle plugin (fork of Fabric Loom) that adds multiplatform build support |
| [Architectury Plugin](https://github.com/architectury/architectury-plugin) | Gradle plugin that processes `@ExpectPlatform` annotations and wires subprojects together |

<Note>
  Architectury API is **optional**. You can build a multiplatform mod using only Architectury Loom and Architectury Plugin. Add this API when you want the abstracted events, registries, and networking it provides.
</Note>

## Platforms supported

This version of Architectury API targets **Minecraft 1.19.2** and supports two mod loaders:

- **Fabric** — requires Fabric Loader ≥ 0.14.0 and Fabric API ≥ 0.66.0
- **Forge** — requires Forge 43.2.0+

## `@ExpectPlatform`

`@ExpectPlatform` (from [Architectury Injectables](https://github.com/architectury/architectury-plugin)) lets you declare a `static` method in your common module and supply separate implementations for Fabric and Forge. Architectury Plugin replaces the method body at compile time so the correct platform implementation is called at runtime.

```java
// common module
public class MyPlatformHelper {
    @ExpectPlatform
    public static String getPlatformName() {
        throw new AssertionError(); // replaced at compile time
    }
}
```

See [ExpectPlatform](/concepts/expect-platform) for the full explanation and naming conventions.
